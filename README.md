### 🔭 Proyecto Principal — Fitness Tracker App

Una aplicación mobile de seguimiento de entrenamientos construida de cero como proyecto de portafolio completo.

- **API REST** con Java 21 + Spring Boot 3 + MySQL
- **Autenticación JWT** implementada con Spring Security
- **Arquitectura en capas** — Controller → Service → Repository → DTO
- **Frontend** React Native con sistema de temas
- **Cloud** desplegado en AWS EC2 con RDS (en progreso)
